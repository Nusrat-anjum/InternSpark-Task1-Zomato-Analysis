# Zomato Dataset Analysis

## Objective
Analyze the Zomato restaurant dataset to find insights about ratings, cuisines, restaurant locations, online ordering, and approximate cost.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Google Colab

## Analysis Performed
- Data Cleaning
- Missing Value Handling
- Duplicate Removal
- Top Restaurant Locations
- Rating Distribution
- Popular Cuisines
- Online Order Analysis
- Cost Distribution
- Correlation Heatmap
